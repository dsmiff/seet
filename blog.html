Fetching GitHub Pull Requests

10th November 2016

Aim is to easily retrieve pull requests made to the repository's base branch. Find the .git/config file, then note the structure inside:

[remote "origin"]
fetch = +refs/heads/*:refs/remotes/origin/*
url = git@github.com:repo/repo.git

Add the line fetch = +refs/pull/*/head:refs/remotes/origin/pr/* to this section.
Now you should fetch the pull request from the repository. Following which, you can check out a particular pull request via git checkout pr/001
